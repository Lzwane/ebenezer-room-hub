# 🏢 Ebenezer Room Hub

A modern web application built to streamline room management, listings, and rentals. This platform provides an intuitive, high-performance interface for users to browse, manage, and book spaces seamlessly.

---

## 🚀 Features

*   **Modern Dashboard:** Clean and responsive UI built for both desktop and mobile users.
*   **Room Management:** Powerful rental store state management to track bookings, availability, and room properties.
*   **Comprehensive UI Components:** Out-of-the-box support for interactive components like carousels, data tables, dialogs, and robust forms.
*   **Global Error Capture:** Integrated client-side and server-side tracking to ensure smooth user experiences.
